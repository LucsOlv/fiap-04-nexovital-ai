# Exemplo de sinais vitais

Arquivo `sample-vital-signs.csv` com 30 registros sintéticos de demonstração.

## Propósito

Ilustrar o fluxo de upload, parsing e detecção de anomalias do pipeline de
`VITAL_SIGNS` do NexoVital AI.

## Origem dos dados

Os valores foram gerados sinteticamente, inspirados em padrões gerais de
sinais vitais adultos. Não representam pacientes reais nem dados do
PhysioNet.

## Limitação e uso

Este arquivo destina-se **exclusivamente à demonstração do pipeline de
processamento**. Os resultados (findings, gráficos e estatísticas) são
evidências para revisão humana e **não configuram diagnóstico médico**.
