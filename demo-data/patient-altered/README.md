# patient-altered

Paciente fictício para demonstração de condição alterada.

Este caso mostra padrão de desconforto respiratório:
- SpO2 cai progressivamente de 97 para 88 ao longo de algumas horas;
- frequência respiratória sobe de 16 para 28;
- frequência cardíaca acompanha piora com elevação gradual;
- áudio esperado traz pausas, frases curtas e queixa de cansaço;
- vídeo esperado mostra baixa amplitude e assimetria de movimento.

Resultado demonstrativo esperado: convergência multimodal com nível `ALERTA`.
