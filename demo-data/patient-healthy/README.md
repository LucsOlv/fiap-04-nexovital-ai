# patient-healthy

Paciente fictício para demonstração de padrão saudável.

Este caso mantém estabilidade clínica:
- sinais vitais dentro das faixas esperadas;
- SpO2 estável em 98-99;
- frequência respiratória entre 15 e 16;
- fala fluida e sem pausas relevantes;
- vídeo com movimento estável e simétrico.

Resultado demonstrativo esperado: análise com nível `NORMAL`.
