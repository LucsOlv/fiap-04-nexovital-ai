# patient-neuro-no-history

Paciente fictício com condição neurológica em investigação e sem histórico prévio.

Características do caso:
- não possui CSV de sinais vitais anterior;
- não possui prescrições prévias para comparação;
- baseline individual indisponível;
- análise deve destacar limitações e evitar interpretar ausência de dados como normalidade.

Resultado demonstrativo esperado: análise parcial, com `ATENÇÃO` ou resultado limitado conforme mídia enviada.
